# Solingo

Learn crypto, Solana & memecoins safely and responsibly.

An educational app that explains blockchain, wallets, security, DeFi and memecoins step by step — so even complete beginners can finally understand before doing anything.

## Links

- **Website:** [solingo.fun](https://solingo.fun/)
- **X (Twitter):** [@learnsolingo](https://x.com/learnsolingo)

## Features

- 60+ micro-lessons, straight to the point
- 6+ tracks (Basics → Solana → DeFi → Memecoins)
- Spaced repetition for better learning
- Progress tracking
- Beautiful UI/UX

## Tech Stack

- React Native / Expo
- TypeScript
- Supabase (Backend)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file based on `.env.example`
4. Start the app:
   ```bash
   npm start
   ```
